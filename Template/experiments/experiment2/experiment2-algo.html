Push Operation

  1. Overflow?
    IF Top = Max - 1 then
    Print: Overflow and Return
  2. Increase Top by 1
    SET Top = Top + 1
  3. Assign Element at Top position
    SET Stack[Top] = Element
  4. Exit

Pop Operation

  1. Underflow
    IF Top = -1 then
    Print: Underflow and Return
  2. Assign Top element to Variable
    SET Element = Stack[Top]
  3. Decrease Top by 1
    SET Top = Top - 1
  4. Exit
