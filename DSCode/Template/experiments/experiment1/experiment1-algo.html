Algorithm: Tower of Hanoi

Introduction

The solution involves breaking down the problem into smaller steps using recursion. Let’s first understand how to solve it with fewer disks.

For 1 Disk:

Move the disk directly from the source to the destination.

For 2 Disks:

  • Move the smaller disk to the auxiliary pole.
  • Move the larger disk to the destination pole.
  • Move the smaller disk from the auxiliary pole to the destination pole.

Recursive Algorithm:

Procedure:

  • Base Case: If there is only 1 disk, move it from the source to the destination.
  • Recursive Case:
    • Move the top n−1 disks from the source to the auxiliary pole.
    • Move the nth (largest) disk from the source to the destination.
    • Move the n−1 disks from the auxiliary pole to the destination.

Pseudocode:

START
Procedure Hanoi(disk, source, dest, aux)
    IF disk == 1, THEN
        move disk from source to dest
    ELSE
        Hanoi(disk - 1, source, aux, dest)   // Step 1
        move disk from source to dest        // Step 2
        Hanoi(disk - 1, aux, dest, source)   // Step 3
END Procedure
STOP
