Experiment 3: Queue Operations

Problem Overview:

A queue is a linear data structure that follows the FIFO (First In, First Out) principle. Elements are inserted from the rear end and deleted from the front end. In this experiment, we will implement a queue using an array in C and perform basic operations such as insertion, deletion, and display.

Algorithm to Initialize the Queue:

  1. [Initialize Queue] Set FRONT = -1 and REAR = -1.
  2. Exit.

Algorithm to Insert an Element in the Queue (enqueue):

  1. [Overflow Check] If REAR = MAX - 1, then:
    • Print "Overflow" and exit.
  2. [Update REAR] If FRONT = -1 and REAR = -1, then:
    • Set FRONT = 0 and REAR = 0.
    Else:
    • Increment REAR = REAR + 1.
  3. [Insert Element] Set QUEUE[REAR] = ITEM.
  4. Exit.

Algorithm to Remove an Element from the Queue (dequeue):

  1. [Underflow Check] If FRONT = -1 or FRONT > REAR, then:
    • Print "Underflow" and exit.
  2. [Remove Element] Assign VAL = QUEUE[FRONT].
  3. If FRONT = REAR, then:
    • Set FRONT = -1 and REAR = -1 (queue is now empty).
    Else:
    • Increment FRONT = FRONT + 1.
  4. [Return Deleted Element] Return VAL.
  5. Exit.

Algorithm to Return the First Element in the Queue (top):

  1. [Empty Check] If FRONT = -1 or FRONT > REAR, then:
    • Print "Underflow" and exit.
  2. [Return Front Element] Return QUEUE[FRONT].
  3. Exit.

Algorithm to Display the Queue Elements:

  1. [Empty Check] If REAR = -1, then:
    • Print "Empty queue" and exit.
  2. [Display Elements] For i = FRONT to REAR, do:
    • Print QUEUE[i].
  3. Exit.
