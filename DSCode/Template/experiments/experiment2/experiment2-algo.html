Algorithm: Stack Operations

Introduction

A stack is a linear data structure that follows the LIFO (Last In, First Out) principle. This experiment demonstrates the implementation of the following operations:

  • Push: Add an element to the stack.
  • Pop: Remove the top element from the stack.
  • Peek: View the top element without removing it.
  • isEmpty: Check if the stack is empty.
  • isFull: Check if the stack is full.
  • Display: Display all stack elements.

Algorithms

Push Operation

  • Step 1: Check for overflow. If Top = Max - 1, print "Overflow" and exit.
  • Step 2: Increment Top by 1.
  • Step 3: Add the element to Stack[Top].

Pop Operation

  • Step 1: Check for underflow. If Top = -1, print "Underflow" and exit.
  • Step 2: Store the top element (Stack[Top]) in a variable.
  • Step 3: Decrement Top by 1.

Peek Operation

  • Step 1: Check for underflow. If Top = -1, print "Underflow" and exit.
  • Step 2: Return Stack[Top].

isEmpty Operation

  • Step 1: If Top = -1, return true (1).
  • Step 2: Else, return false (0).

isFull Operation

  • Step 1: If Top = Max - 1, return true (1).
  • Step 2: Else, return false (0).

Display Operation

  • Step 1: If Top = -1, print "Stack is empty" and exit.
  • Step 2: Iterate from Top to 0 and print each Stack[i].
